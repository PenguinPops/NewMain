C++ App Development with the Electron Framework

I have created a program called FormatC++ that aims to simplify code formatting for C++ developers. It is designed as a user-friendly Electron app with an intuitive interface. The main purpose of FormatC++ is to take unformatted code containing enumerations and transform it into a format that can be easily copied and used in any development environment.

Currently, FormatC++ offers the following functionalities:

  • Recognition and removal of numbers and spaces before the code.
  • Identification of brackets and appropriate formatting.
  • Deletion of empty lines for a cleaner code appearance.
With FormatC++, you can quickly and effortlessly format your C++ code, making it more readable and organized.
